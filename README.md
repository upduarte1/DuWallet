# DuWallet  
Android App for Monthly Expense Tracking

## Overview  
**DuWallet** is a simple and intuitive Android application designed to help users monitor and control their monthly expenses.  
It allows you to record transactions, view monthly summaries, and improve your personal finance management.

## Key Features  
- Add new expenses (amount, category, date)  
- View a monthly summary of all expenses  
- Clean and user-friendly interface  
- Built using Java for Android
